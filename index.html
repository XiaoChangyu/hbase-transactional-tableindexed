HBase-trx by Troove Inc.

An extension of HBase that enables tables to have:
  • Index Tables - Index your table with a customizable key structure to optimize scan queries.
  • Application-level Transactions - Guarantee serialization and atomicity of HBase operation sequences using JTA.

Troove is developing this open-source project to meet the requirements of it's internal projects. Please feel free to jump in and help us to mature the code into a solid general purpose tool.

Dependencies
